# Weather in St. Petersburg 🌤️

Веб-сайт для отображения текущей погоды в Санкт-Петербурге, созданный с помощью ИИ-агента. Интерфейс полностью на русском языке.

## 🚀 Особенности
*   **ИИ-разработка**: Код полностью сгенерирован ИИ-агентом.
*   **Актуальные данные**: Отображение точной погоды в Санкт-Петербурге в реальном времени.
*   **Локализация**: Полная поддержка русского языка.
*   **Адаптивность**: Сайт отлично выглядит на смартфонах, планшетах и ПК.
*   **Поддержка светлой и тёмной темы**

## 🛠️ Технологии
*   HTML5 / CSS3 / JavaScript
*   Open-Meteo API

### Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```

2. Перейдите в папку проекта:
   ```bash
   cd имя-репозитория
   ```

3. Запустите проект:
Просто откройте файл `index.html` в вашем браузере.

## 🤖 Как это было создано
Этот проект — результат демонстрации возможностей автономных ИИ-агентов. Агент самостоятельно спроектировал структуру, написал логику запросов к API погоды и сверстал интерфейс по текстовому описанию (промпту).
